<h1>Blog con Jekyll</h1>
<p>Este proyecto consiste en la creación de un blog estático utilizando Jekyll, HTML, CSS y Sass.</p>

<ul>
  <li>Diseño personalizado con Sass</li>
  <li>Uso de layouts e includes para modularidad</li>
  <li>Formulario de contacto funcional</li>
  <li>Publicación en GitHub Pages</li>
</ul>

<p>Este blog está pensado para desarrolladores que quieren compartir contenido técnico de forma organizada y elegante.</p>

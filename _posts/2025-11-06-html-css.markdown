---
title: Aplicando HTML y CSS en Ingeniería Informática
---

HTML (HyperText Markup Language) y CSS (Cascading Style Sheets) son los pilares fundamentales del desarrollo web moderno. HTML proporciona la estructura básica de las páginas, permitiendo organizar el contenido en elementos como encabezados, párrafos, listas, enlaces, imágenes y formularios. Por su parte, CSS se encarga de la presentación visual, definiendo estilos como colores, tipografías, márgenes, alineaciones y animaciones. Juntos, forman el núcleo de cualquier sitio web, desde los más simples hasta los más complejos. En el contexto de la Ingeniería Informática, dominar HTML y CSS no solo permite crear interfaces atractivas, sino también comprender cómo se comunican los navegadores con el código fuente, cómo se renderiza el contenido y cómo se optimiza la experiencia del usuario. Además, el uso de buenas prácticas en HTML semántico y CSS modular contribuye a mejorar la accesibilidad, el rendimiento y la mantenibilidad del sitio. Estas tecnologías también sirven como base para aprender frameworks más avanzados como Bootstrap, Tailwind CSS o incluso sistemas de diseño personalizados que se integran en aplicaciones web dinámicas.

En este blog, HTML y CSS han sido utilizados para estructurar contenido técnico de manera clara y visualmente agradable. Cada entrada está diseñada para facilitar la lectura, destacando conceptos clave mediante estilos personalizados y una jerarquía visual coherente. La navegación entre secciones como “Inicio”, “Sobre mí”, “Contacto” y “Proyectos” se ha construido con HTML bien organizado y estilizado con CSS para lograr una experiencia fluida. Las imágenes, los enlaces y los bloques de texto se han dispuesto estratégicamente para reforzar el mensaje de cada publicación. Además, se han aplicado estilos responsivos para garantizar que el blog se visualice correctamente en distintos dispositivos, desde ordenadores hasta teléfonos móviles. Esta implementación demuestra cómo HTML y CSS pueden ser herramientas poderosas no solo para mostrar información, sino también para comunicar ideas de forma efectiva en el ámbito académico y profesional. A través de este proyecto, se evidencia que el diseño web no es solo una cuestión estética, sino una disciplina que combina lógica, creatividad y precisión técnica.
